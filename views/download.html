  • 01. Install node.js platform

    Firstly you must install node.js platform. Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient, perfect for data-intensive real-time applications that run across distributed devices.

  • 02.2 Node Package Manager

    You can install / download framework from Node Package Manager. Node Package Manager is automatically installed with node.js platform. It's easy and fast. Create empty directory and open terminal (Mac OS X, Linux) or command line (Windows) and write below command:


    $ npm install total.js

  • 02.3 Run it without any installation

    You can run framework without any installation. Create e.g. index.js in project directory and run it through terminal / command-line node index.js


    // Execute in node.js
// Framework automatically reads directory structure
var http = require('http');

// This is total.js - web application framework for node.js
http.get('http://modules.totaljs.com/total.js/v1.7.0/total.js', function(res) {
    var buffer = [];
    res.on('data', function(buf) { buffer.push(buf.toString('utf8')); });
    res.on('end', function() { eval(buffer.join('')); if (typeof(init) !== 'undefined') init(); });
});

// Initialize function
// Optional
function init() {

    framework.route('/', function() {
        this.plain('Hello world!');
    });

    // debug | release | test
    framework.http('debug');
}
