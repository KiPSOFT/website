TotalStack

Framework supports great features for development Angular.js applications.

  • TotalStack is a module with group of functions which can help you to create an Angular.js application easily. The module supports "extra" one virtual directory for Angular.js application. TotalStack offers great server side features with good directory structure for Angular.js application. All settings are configurable.


    total.js command-line for Angular.js



Create an angular.js application from command-line

# --> Install total.js as global module
# --> NPM installs executable file "total"
$ sudo npm install -g total.js

# --> Create App directory
$ mkdir angularapp

# --> Open App directory
$ cd angularapp

# --> Create empty angular.js project
$ total -angular

# --> Run App
$ node debug.js
