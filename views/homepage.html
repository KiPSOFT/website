Web framework for node.js

Free server framework for building Web sites and Web applications using JavaScript, HTML and CSS.


Modern framework

Friendly URL, XHR, JSON, LESS CSS (CSS 3), JavaScript minifier, XSS protected, resources, modules and supports MVC architecture.

WebSockets (RFC 6455)

WebSocket is an advanced technology that makes interactive communication session between the user's browser and a server.

No dependencies

The framework has no dependencies. Complete functionality is built in its core. You can install any module by the Node Package Manager.

Gives you freedom

Framework doesn't limit you. You can write whatever you want or rewrite existing functionality. It's easy, simple and safe.

Total.js supports great features for development Angular.js applications. I call this: TOTALSTACK.
Try to create Angular.js application with total.js web application framework for node.js from command-line tool:

$ sudo npm install -g total.js
$ mkdir angularapp
$ cd angularapp
$ total -angular
$ node debug
